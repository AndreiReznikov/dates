## Запуск проекта

### Требования:

- Node.js >= 18.x
- npm >= 9.x

### Установка

1. Клонируйте репозиторий и перейдите в его корневую директорию:

```bash
  git clone https://github.com/AndreiReznikov/dates
```

2. Установите зависимости:

```bash
  npm install
```

3. Для запуска используйте команду:

- npm run start — локальный запуск в режиме разработки
